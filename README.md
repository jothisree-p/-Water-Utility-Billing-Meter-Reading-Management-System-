Water Utility Billing & Meter Reading Management System (JDBC)

Project Description

The Water Utility Billing & Meter Reading Management System is a console-based Java application developed using JDBC to automate water consumer management, meter reading records, and monthly bill generation.
The system ensures data accuracy, transactional integrity, and rule-based billing, replacing manual billing processes used by water utility departments.

Objectives

Manage water consumers efficiently
Record monthly meter readings without duplication
Generate accurate water bills based on consumption
Maintain outstanding balances
Support bill cancellation with balance restoration
Ensure database consistency using JDBC transactions

Technologies Used

Java (Core Java)
JDBC
Oracle Database
SQL
DAO Design Pattern
Console-based UI
